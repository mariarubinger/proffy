## Prerequisites

- [Node LTS](https://nodejs.org/en/)
- [Yarn](https://classic.yarnpkg.com/pt-BR/) or [npm](https://www.npmjs.com/)
- [Expo](https://docs.expo.io/)

## Cloning project

```bash

  # Clone repository
  $ git clone https://github.com/mariarubinger/proffy-web

  # Enter folder
  $ cd proffy-web
  
  ```

## Install back-end

```bash

  # Enter folder
  $ cd server

  # Install dependences
  $ yarn

  #Execute
  $ yarn start

```

## Install front-end

```bash

  # Enter folder
  $ cd web

  # Install dependences
  $ yarn

  #Execute
  $ yarn start

```
